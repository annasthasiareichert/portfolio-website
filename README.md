# Portfolio-Website — Annasthasia Reichert

Persönliche Portfolio-Website von **Annasthasia Reichert**, Grafik- & UI-Designerin (Köln).
Editorial, ruhig, mit gezielter Bewegung — zweisprachig (Deutsch / Englisch).

## Aktueller Stand

Frühe Entwurfsphase. Der erste sichtbare Baustein ist ein visueller Entwurf der Startseite:

- **`entwurf/startseite.html`** — eigenständige HTML-Seite, per Doppelklick im Browser ansehbar
  (kein Build nötig). Zeigt Namens-Lockup, Arbeiten-Index mit Hover-Bild-Reveal, Bewegungsmomente
  und den DE/EN-Umschalter. Bilder sind noch Platzhalter.

Der visuelle Stil leitet sich aus Annasthasias Lebenslauf ab (Greige/Weiß/Schwarz, Schreibschrift +
fette Grotesk + Serifen-Kursiv).
